If you're using a Trial Twilio account, you can only send SMS to verified phone numbers.

How to Verify Your Phone Number:

  1. Go to Twilio Console ‚Üí Phone Numbers ‚Üí Verified Caller IDs
  2. Click "Add a new Caller ID"
  3. Enter your phone number (e.g., +17402581804)
  4. Twilio will send you a verification code via SMS or call
  5. Enter the code to verify your number
  6. Once verified, try signing up again!

Alternative: Upgrade Your Twilio Account

If you upgrade to a paid Twilio account, you can send SMS to any number without verification.
